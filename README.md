# 👕 Mig Casual Clothes - Landing Page

Uma landing page de moda conceitual e responsiva, desenvolvida com um design minimalista e transições dinâmicas entre seções. O projeto destaca coleções sazonais utilizando sliders verticais e uma navegação fluida.

## 🚀 Demonstração

Este projeto apresenta uma interface dividida em três colunas dinâmicas:
- **Banner Principal:** Título e chamada para ação (CTA).
- **Banner Central:** Exibição de produtos via Swiper.js (Sliders verticais infinitos).
- **Banner Lateral:** Conteúdo editorial e blog.

## 🛠️ Tecnologias Utilizadas

*   **HTML5** - Estrutura semântica.
*   **CSS3** - Estilização moderna com variáveis (Custom Properties) e Flexbox/Grid.
*   **JavaScript (Vanilla)** - Lógica de transição de estados e manipulação de classes.
*   **[Swiper.js](https://swiperjs.com/)** - Biblioteca para os sliders de imagens.
*   **[Ionicons](https://ionic.io/icons)** - Pack de ícones utilizados na interface.
*   **Google Fonts** - Fontes *Lobster Two*, *Noto Sans* e outras.

## 🎨 Funcionalidades Principais

*   **Navegação Alternada:** Ao clicar em "Browse", o layout se contrai para revelar o catálogo e o blog.
*   **Sliders Sincronizados:** Três sliders verticais independentes que rodam automaticamente em velocidades e direções diferentes.
*   **Menu Mobile Adaptativo:** Menu lateral estilizado que aparece apenas no modo de navegação expandida.
*   **Design Responsivo:** Adaptado para diferentes resoluções de tela.<hr>
# 👕 Mig Casual Clothes - Landing Page

A conceptual and responsive fashion landing page, developed with a minimalist design and dynamic transitions between sections. This project showcases seasonal collections using vertical sliders and fluid navigation.

> **Note:** This project was developed for educational purposes.

## 🚀 Live Demo

The project features an interface divided into three dynamic columns:
- **Main Banner:** Title and Call to Action (CTA).
- **Middle Banner:** Product showcase via Swiper.js (Infinite vertical sliders).
- **Right Banner:** Editorial content and blog section.

## 🛠️ Tech Stack

*   **HTML5** – Semantic structure.
*   **CSS3** – Modern styling using Custom Properties (variables) and Flexbox/Grid.
*   **JavaScript (Vanilla)** – Transition logic, state management, and class manipulation.
*   **[Swiper.js](https://swiperjs.com/)** – Library used for the image sliders.
*   **[Ionicons](https://ionic.io/icons)** – Icon pack used across the interface.
*   **Google Fonts** – Featuring *Lobster Two*, *Noto Sans*, and *Raleway*.

## 🎨 Key Features

*   **Interactive Navigation:** Clicking "Browse" triggers a layout transition that reveals the catalog and blog.
*   **Synchronized Sliders:** Three independent vertical sliders running at different speeds and directions (upward/downward).
*   **Adaptive Mobile Menu:** A custom side menu that appears only when the navigation is in its expanded state.
*   **State Management:** The JS ensures that the mobile menu automatically closes when returning to the desktop view.
